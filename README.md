# Jailbreak-Jailbreak

The dataset is quite small, and will be updated with around 600 new jailbreaks very soon!

The Current supported Jailbreaks That the AI uses in its dataset: `LittleGPT`, `Maximum`, `Leo`, and `Devils Advocate`.
